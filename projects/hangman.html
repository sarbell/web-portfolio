Hangman - Javascript game

Previous Next
Details:

This was a project for my beginning JavaScript course. It entailed using pure JavaScript to create an interactive game. We were given about 2 weeks to create the game, and the goal was to make a game that was visually appealing and simple to use.

Technologies:

Html 5, Css3, javascript

See Finished Product
Summary:

When I first got the assignment to create a game out of JavaScript I was unsure of what direction to go. I chose to do Hangman, the classic word guessing game, because it would require a lot of work with event listeners, array manipulation, and creative design. I feel like I learned a lot from this project because it allowed me to practice with DOM manipulation to create something that was fun to play.

I started my process my first utilizing the console, and creating the hangman game there. I then built the interface, and then worked on connecting the two pieces so it would be completely functional. The first step was to choose a word to be guessed randomly from the word bank array, have a copy of that word displayed as dashes, then have the users guess traverse the word to find the indexes that the character was the same. I also implemented checks to not allow the same letter to be guessed or numbers or symbols to be used. The program would keep track of the score and display the hangman drawing with missed letters.

The design was inspired by the “old west” with the scenery, colors and typeface used. Overall, the hardest part of this project was displaying the CSS correctly. At the time of the project, I had very little experience with CSS, so making the hangman drawing look the same on all screens was a challenge.
